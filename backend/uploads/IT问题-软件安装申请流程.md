# 软件安装申请流程

## 适用范围

本文档适用于员工因工作需要申请安装新软件，包括办公软件、专业工具软件、开发工具等。

## 非标软件申请流程

### 第一步：提交申请
1. 登录 OA → IT 服务 → 软件安装申请
2. 填写以下信息：
   - 软件名称和版本要求
   - 用途说明（解决什么工作需求）
   - 是否需要购买许可证（付费软件注明预算）

### 第二步：审批流程
- 免费软件（如 VS Code、Postman）：部门主管审批 → IT 安全评估 → 安装
- 付费软件（如 Photoshop、JetBrains）：部门主管 → 部门总监 → 采购 → IT 安装
- 审批周期：免费软件 1-2 个工作日，付费软件 5-10 个工作日

### 第三步：安装方式
- 有管理员权限的设备：IT 提供安装包和指引，员工自行安装
- 无管理员权限的设备：IT 远程安装
- 需要特殊配置的软件：IT 上门安装

## 常见问题

**Q：能不能自己下载安装？**
A：公司设备默认禁止自行安装软件。需通过 OA 申请流程，IT 评估安全性后安装。

**Q：安装失败怎么办？**
A：提交工单，附上安装报错截图，IT 会远程协助。

**Q：软件许可证过期怎么续费？**
A：联系 IT 登记，由 IT 统一管理许可证续费。

**Q：同一个软件能安装在多台设备上吗？**
A：取决于许可证类型。部分许可证限定单设备，需额外购买。
